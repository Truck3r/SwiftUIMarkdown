## Reporting a Vulnerability

If you discover a security vulnerability within **MarkdownRenderer**, please follow these steps:

1. **Do not open a public GitHub issue.**
2. Instead, email the maintainer directly at **[Sune.Riedel@gmail.com]**.
3. Include:
   - Description of the vulnerability
   - Steps to reproduce (if applicable)
   - Suggested fix or mitigation

You will receive a response within 72 hours. Once the issue is confirmed, a patch release will be prepared and credited appropriately.
